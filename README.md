# B4511, Блок - Python


CTRL + S - сохранения файла
CTRL + Z - шаг назад
CTRL + A - выделения всего
End - перемещение курсора в конец строки
Home - перемещение курсора в начало строки 
Tab - два пробела 
Shift + Tab - два пробела в левую сторону
CTRL + L - открытие поисковика по файлу
CTRL + D - дублирование строки (на которой стоит курсор)
CTRL + X - удаление строки (на которой стоит курсор)
