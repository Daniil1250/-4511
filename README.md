# Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `CTRL + S` | Сохранение файла |
| `CTRL + Z` | Шаг назад (отмена действия) |
| `CTRL + A` | Выделение всего |
| `End` | Перемещение курсора в конец строки |
| `Home` | Перемещение курсора в начало строки |
| `Tab` | Вставить два пробела |
| `Shift + Tab` | Удалить два пробела влево |
| `CTRL + L` | Открытие поиска по файлу |
| `CTRL + D` | Дублирование текущей строки |
| `CTRL + X` | Удаление текущей строки |
